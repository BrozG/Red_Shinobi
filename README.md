# RED SHINOBI

> Multi-Agent AI Terminal with Model Context Protocol Support

## Installation

```bash
cd red_shinobi
pip install -e .
```

## Usage

Start the CLI:
```bash
red
```

## Commands

- `/key` - Configure API keys
- `/models` - View available models
- `/mcp <name> <command>` - Connect to MCP servers
- `/file <path> <query>` - Query files with AI
- `/system <model> <prompt>` - Update system prompts
- `/save` - Save conversation transcript
- `/chatbox` - Launch graphical UI
- `/help` - Show help
- `/exit` - Exit
